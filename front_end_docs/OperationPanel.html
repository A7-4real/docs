Class: OperationPanel

OperationPanel(options)

new OperationPanel(options)

A operation panel that reads the single or multiple JSON schema and generates the data form and form control. Dependency: lib/pureform/pure-form.js lib/pureform/pure-form.css
Parameters:
Name Type Attributes Default Description
options Object All required and optional settings for instantiating a new instance of a operation panel.
options.id String <optional>
 The container id for operation panel. The all panel'instance will be stored into this.elt if the id empty.
options.zIndex String <optional>
 600 The z index of Instance on webpage.
options.title String <optional>
 '' The title of the select control for changing the data form.
options.action Object <optional>
 The action options to configurate the text, title and action of callback on the action btn
