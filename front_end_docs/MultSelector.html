Class: MultSelector

MultSelector(options)

new MultSelector(options)

A MultSelector that provide multiple selected functionality. Events: 'remove-all', 'remove', 'select-all', 'select', 'cancel', 'action' Example: const mult-selector = MultSelector({id:'test'}); mult-selector.addHandler('remove',function(data){}); mult-selector.removeHandler('select-all',function(data){});
Parameters:
Name Type Description
options Object settings for instantiating a new instance of a mult-selector.
Properties
Name Type Attributes Default Description

 The container id for mult-selector. The mult-selector's instance will be stored into this.elt if the id empty.
element Element <optional>
 The container as a html element.

 The data of selector's options. The form of each option should be an array -> [key - identity,value - text on option].

 The title of mult-selector

 'Action' The text of action btns
