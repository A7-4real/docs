Introduction

Get started with CaMicroscope, truly free, scalable web-based biomedical image and data viewer, with strong emphasis for viewing, annotating, and analyzing biomedical images.

CaMicroscope Front End use OpenSeadragon and Material Icons as dependencies. It is easy to use exsiting UI components (such as CaMic, CaMessage, CaToolbar and LayersViewer ) to quick create and integrate a customized digital biomedical imaging view system.

Start
