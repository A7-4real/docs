Class: CollapsibleList

CollapsibleList(options, list, changeCallBackopt)

new CollapsibleList(options, list, changeCallBackopt)

CaMicroscope Collapsible List. the item in this list consists of the item's head and the item's body. Click on the head, if item's body is collapsed then expand it, if not, then expand it. Only one of item can be expanded.
Parameters:

options Object All required and optional settings for instantiating a new instance of a Collapsible List.

 false The content expand or not.
changeCallBack function <optional>
 when the expanded item changes the event is fired. Return {id: item id,isExpand: the state of current item[expand or not]}
