Class: CaToolbar

CaToolbar(options)

new CaToolbar(options)

CaMicroscope Tool Bar. Currently, it shows at the top-left corner of the screen. It consists of Main Tools and Sub Tools. Main Tools is formed of Apps and Layers. There is a callback function that return the status of Main Tools. Sub Tools can be customized by using options.
